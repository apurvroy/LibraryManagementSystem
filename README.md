# LibraryManagementSystem
It's a Library Management System console app using Kotlin.


#Actors
  *Admin
  *Faculty
  *Student

#Functionalities:
1.Adding a Book
2.Removeing a Book
3.Issue a Book
4.Return a Book
5.Renew a Book
6.Fine calculation
7.Add a Student
8.Show all Books
9.Login/Logout
10.Different Types of Books
11.Trackability of status of Each and Every Book
